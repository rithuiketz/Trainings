# Instructions to build and run the Spring Boot REST demo with Java 8

1. Open a terminal and navigate to the project directory:
   cd c:\Rithuik\Trainings\Docker

2. Build the Docker image:
   docker build -f SpringBoot_DockerFile -t springboot-rest-demo .

3. Run the Docker container:
   docker run -p 8080:8080 springboot-rest-demo

4. Access the REST endpoint in your browser or with curl:
   http://localhost:8080/hello

You should see the response: Hello, World!
